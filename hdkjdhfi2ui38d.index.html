Sep 9 2024

New icons!

For the past 2 years or so, I've been using an icon library called Font Awesome. It has several styles, it is very extensive and has almost every icon I needed. The best part was that every icon is fully affected by text modifications like color, size, text-shadow, et cetera. I encourage you to check it out if you're starting a project of your own!
It had its limitations, though, and adding custom icons was only available through paid features. Which is why, starting today, I will be making and using my own svg icon library to get the most out of css, and to create a better user experience. It's very small right now, with only about twenty icons, but as the website grows, the library will grow with it. I hope you like the new icons!
Cheers, wraithcode
